Bringing Privacy to the Masses

Coinjoin Any Amount

Traditional coinjoins require users to have UTXO's of certain amounts, because their privacy is based on equal value outputs. Usually this minimum required amount is fairly high, making it impossible for many to coinjoin. With WabiSabi's' unequal amount coinjoins, there's no minimum amount required.

Unlimited Free Remixes

Whenever user sends a payment with freshly coinjoined coins, him/her and the recipient are able to coinjoin those coins without paying any coordinator fees*. Remixing is also free* as well as coinjoining UTXOs smaller than 1 million satoshi.

Easy to use UI

Wasabi is designed to be a user friendly and intuitive Bitcoin wallet. User doesn't need to worry about coin selection, coinjoining or change avoidance. The software handles everything automatically by default.

*Bitcoin network fees still apply.
